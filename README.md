# Gestão de Vendas e Fechamento (MVPv1.0)
Planilha seguindo lógica de sistemas para gestão do fluxo de vendas e fechamento do mensal. Utilizando lógica de integração de dados e gestão de entidades.

O Problema: Como dono e gestor de uma empresa de venda de Certificado Digital ICP-Brasil, eu precisava centralizar o fluxo de vendas, calcular comissões automáticas para parceiros (contadores) e prever o lucro líquido após impostos e custos fixos.

A Solução: Desenvolvi uma arquitetura de dados em planilhas Excel e Sheets que realiza o cruzamento de tabelas (Vendas x Parceiros x Financeiro) para automatizar o fechamento mensal, podendo ser acessada e modificada de forma online através do Sheets.

O que este projeto demonstra:

Lógica de Negócio: Regras complexas de comissionamento (porcentagens variáveis por parceiro).

Modelagem de Dados: Estrutura de tabelas relacionais (Parceiro -> Venda -> Repasse).

Visão de Produto: Foco total em KPIs (Lucro por certificado, Custo fixo, Margem líquida).
